# Hello React - Front End

> This is a simple react-front-end to consume this [Hello Rails Back End](https://github.com/nicupop729/hello-rails-back-end)

## Built With

- Major languages: JavaScript;
- Frameworks: React;
- Technologies used: Redux, GitHub, GitFlow, Terminal;

## Getting Started

### Prerequisites

- Please make sure you have Node.js packedge installed on your terminal.

### Setup

To get a local copy up and running follow these simple steps.

To setup the bookstore project in your local, in the repo page:
click on code (dropdown list) > Download as ZIP;
or open terminal of path you want to install project and run this command <br>
`git clone git@github.com:nicupop729/hello-react-front-end.git`.

### Install

Run in your terminal the following commands:

**`$ cd hello-react-front-end/`**<br>
**`$ npm install`**<br>
**`$ npm start`**
